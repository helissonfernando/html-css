Background-color - Cor de fundo da pagina

Background-image - Coloca uma imagem no fundo da pagina

OBS: background: linear-gradient(to bottom , rgb)

Background-repeat - Configurar a repetição da imagem

  • Repeat
  • no-repeat
  • repeat-x
  • repeat-y

Background-postition - Cofigura o piscionamento da imagem de fundo

  • Left top = Esquerda e Cima
  • Left center = Esquerda no centro
  • Left Bottom = Esquerda em baixo
  • Right top = Direita em cima
  • Right center = Direita no Centro
  • Right bottom = Direita em baixo
  • Center top = Centro no topo
  • Center center = Centro no centro
  • Center Bottom = Centro em baixo

Background-orgin - Especifica onde será a origem do poscionamento da imagem de fundo

  • Border-box = A limagem inicia a esquerda no topo
  • Padding-box = A imagem fica padrão
  • Content-box = A imagem inicia a esquerda no topo do conteudo do conteiner onde a imagem está aplicada

Background-clip - Especifica a area do fundo

  • Border-box = O fundo inicia a esquerda no topo
  • Padding-box = O fundo inicia a esquerda no topo levando em consideração o padding
  • Content-box = O fundo inicia a esquecerda no topo do conteudo do conteiner

OBS: PARA USAR O Background-clip é preciso colocar as seguites informações 

                Border: 10px solid black;
                Padding: 10px;
                Background: #0f0
                Background-clip: Border-box;

Display - Propriedade indica como um elemento será exibido

Display : block

Um elemento que tem o display configurado como block sempre inicia em uma nova linha não aceita elemntos na mesma linha que ele ou seja quebra de linha após o elemento e sua area ocupa toda a linha onde ele é inserido

Elemento com display:block padrão
<.div.>
<.p.>
<.form.>
<.header.>
<.footer.>
<.section.>
<.table>

Display: inline
Os elementos configurados com display:inline não iniciam em uma nova linha nem quebram de linha após o elemento eles não ocupam a linha inteira, somento o espaço de seu conteúdo.

Exemplos de alguns elementos com Display: inline
<.span>
<.a.>
<.img.>

Display : nome
Este valor para propriedade display pode ser usado para ocultar o elemento pelo javascript

Display : inline-block - Parecido com inline, so que preserva o tamanho configurado do elemento

Display: inline-table Elemento é exibido como uma tablea, não inciam em uma nova linha nem quebra de linha após o elemento, ele não ocupa uma linha intenra somente o espçao do conteudo
<.table.>

Position - A propriedade especifica como um elemento será posicionado na tela
Muitos programadores associam a propriedade position somento ao elemento <.div.> mas a verdade é que podemos utilizar em muitos outros elemento como <.img.> e <.p.>

  • Static - Valor padrão de todos os elemento, os elemento snão podem ser controlado spor top e left e não tem seu posiconamtento afetado.

  • Relative - Tem seu posiconamento left e top relacionado como o elemento anterios

  • Fixed - Neste posiconamento o elemento é posicionado em relação a janela independete se esteja dentro de outro ou não

  • Absolute - Tem seu posicionamento left e top relacionado com o elemento pai e não com o elemento anterior

Float - Esta prorpeidade especifica se o elemento vai flutuar ou não

  • Float : left - posicionar a imagem a esquerda

  • Float : right - posicionar a imagem a direita
